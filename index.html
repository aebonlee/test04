YouTube 영상 제작 프로세스

소재 파일 하나로 영상 재료 완성

롱폼 (10~15분) 숏폼 (1분 이내)
1
소재 준비
영상으로 만들 콘텐츠를 MD 파일로 준비합니다.
이미 작성해둔 콘텐츠 파일을 그대로 사용하거나,
새로운 주제가 있으면 텍스트로 정리해서 파일로 저장합니다.
사용자
2
경쟁 영상 리서치
소재와 비슷한 주제의 인기 유튜브 영상 3개를 자동으로 찾습니다.

찾은 영상마다:
• 자막 전문을 그대로 추출하고
• 조회수, 채널명, 영상 길이 등 기본 정보를 가져오고
• 자막을 AI에게 넘겨서 "이 영상이 왜 인기 있는지", "도입→본론→마무리 구성이 어떻게 되어 있는지"를 분석하고
• 그 구성을 참고해서 우리 영상에 쓸 수 있는 뼈대 구성안을 만들어줍니다.
Gemini (영상 검색 + 분석) yt-dlp (영상 정보) youtube-transcript-api (자막) 자동
3
제목 + 썸네일 + 대본 한번에 생성
보통은 제목 따로, 썸네일 문구 따로, 대본 따로 만들어야 하는데,
여기서는 소재 파일과 2단계 리서치 결과를 합쳐서
세 가지를 한번에(올인원) 생성합니다.

• 영상 제목 — 클릭을 유도하는 제목
• 썸네일 문구 — 썸네일 이미지에 들어갈 텍스트
• 전체 대본 — 영상에서 읽을 전체 스크립트

생성 후 사용자가 확인하고, 수정이 필요하면 수정합니다.
Claude (대본 작성) 사용자 확인/수정
여기서부터 자동 생성
4
대본을 장면별로 분리
확정된 대본을 분석해서 장면(섹션)별로 자동 분리합니다.

왜 필요한가?
• 장면마다 맞는 이미지를 따로 만들어야 하고
• 장면마다 음성 파일을 따로 생성해야 하고
• 나중에 영상 편집할 때 장면 순서와 타이밍을 맞춰야 하기 때문입니다.
자동
5
장면별 이미지 생성
각 장면의 내용에 맞는 이미지 생성용 프롬프트를 자동으로 만듭니다.

생성된 프롬프트를 Whisk 또는 Nana Banana 같은
AI 이미지 생성 도구에 붙여넣어서 이미지를 만듭니다.
프롬프트 자동 생성 사용자가 이미지 도구에서 생성
6
음성(나레이션) 생성
각 장면의 대본 텍스트를 ElevenLabs TTS API에 보내서
사람처럼 자연스러운 음성 파일(MP3)을 장면별로 생성합니다.

내 목소리를 복제(Voice Cloning)해서 사용할 수도 있습니다.
ElevenLabs API 자동
7
유튜브 업로드 정보 생성
유튜브에 영상을 올릴 때 필요한 정보를 자동으로 만들어줍니다.

• 영상 제목 (3단계에서 만든 것)
• 영상 설명 — 키워드 포함, SEO 최적화
• 태그 — 검색에 잘 걸리는 해시태그
• 편집 가이드 — 장면별 타이밍과 순서 정리

이 정보를 보면서 영상 편집과 업로드를 진행합니다.
자동
8
영상 편집 → 유튜브 업로드
지금까지 만든 재료들을 Vrew(영상 편집 프로그램)에서 조합합니다.

• 음성 파일을 타임라인에 배치하고
• 각 장면에 맞는 이미지를 올려놓고
• 자막을 달면 영상 완성

완성된 영상을 유튜브에 업로드하고,
7단계에서 만든 제목/설명/태그를 입력합니다.
Vrew (편집) YouTube (업로드) 사용자
9
블로그 글 생성 (선택)
영상 업로드가 끝나면, 원본 소재 내용에
유튜브 영상 URL을 붙여서 블로그 글을 자동 생성합니다.

소재 자체가 이미 잘 정리된 콘텐츠이므로,
여기에 영상 링크만 추가하면 블로그 글이 됩니다.
자동 (선택)
